# 测试计划

## 概述

本文档定义了本项目的测试策略与计划。

## 测试金字塔

```
          E2E
        /    \
      API    UI
     /   \
  Unit  Integration
```

## 测试目标

| 层级 | 覆盖率目标 |
|------|-----------|
| 单元测试 | ≥80% |
| 集成测试 | ≥60% |

## 测试框架

- **Java**: JUnit 5 + Mockito
- **Go**: testing + gomock

## 测试环境

- 独立测试数据库
- 测试前后数据清理
- 使用事务回滚

## 测试用例

### 单元测试
- 服务层逻辑测试
- 工具函数测试

### 集成测试
- API接口测试
- 数据库操作测试

### E2E测试
- 端到端流程验证
