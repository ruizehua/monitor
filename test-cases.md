# 测试用例文档

## 概述

本文档定义了监控系统的关键测试用例。

## 测试用例清单

### 一、客户端注册测试

#### 1.1 正常注册

| 项目 | 内容 |
|------|------|
| 测试ID | TC-CLIENT-001 |
| 测试名称 | 注册新客户端 |
| 前置条件 | 服务端已启动 |
| 测试步骤 | 1. 发送POST /api/client/register<br>2. 检查响应状态码 |
| 请求体 | `{"clientName": "test-client", "hostIp": "192.168.1.100"}` |
| 预期结果 | 返回200，包含clientId和监控配置 |

#### 1.2 重复注册

| 项目 | 内容 |
|------|------|
| 测试ID | TC-CLIENT-002 |
| 测试名称 | 注册已存在的客户端 |
| 前置条件 | 客户端已注册 |
| 测试步骤 | 1. 发送POST /api/client/register<br>2. 使用相同clientName |
| 预期结果 | 返回200，更新现有客户端信息 |

#### 1.3 参数缺失

| 项目 | 内容 |
|------|------|
| 测试ID | TC-CLIENT-003 |
| 测试名称 | 缺少必填参数 |
| 测试步骤 | 1. 发送POST /api/client/register<br>2. 不提供clientName |
| 预期结果 | 返回400错误 |
