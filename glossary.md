# 术语表

## 概述

本文档定义了项目中的核心术语和概念。

## 术语定义

| 术语 | 定义 |
|------|------|
| **服务端** | Java后端服务，负责客户端管理和数据存储 |
| **客户端** | Go客户端程序，负责主机资源采集和上报 |
| **监控项** | 需要监控的资源指标（CPU、内存、磁盘、进程） |
| **上报间隔** | 客户端上报数据的时间间隔（默认30秒） |
| **监控数据** | 客户端采集的主机资源信息 |
| **注册** | 客户端首次连接服务端的过程 |

## 缩写说明

| 缩写 | 全称 | 说明 |
|------|------|------|
| CPU | Central Processing Unit | 中央处理器 |
| RAM | Random Access Memory | 随机存取内存 |
| API | Application Programming Interface | 应用程序编程接口 |
| REST | Representational State Transfer | 表述性状态传递 |
| JWT | JSON Web Token | JSON Web令牌 |
